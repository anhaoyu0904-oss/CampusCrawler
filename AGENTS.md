## Highest-priority legal and safety boundary

This project may collect only information that is publicly accessible on normal web pages.

Never implement or perform any of the following:

- Bypassing login, authentication, paywalls, access controls, IP restrictions, or other authorization boundaries.
- Bypassing CAPTCHAs, anti-bot challenges, rate limits, robots restrictions, or technical protections.
- Using stolen, leaked, shared, or otherwise unauthorized accounts, cookies, tokens, credentials, or private APIs.
- High-frequency or large-scale scraping that could disrupt a website or exceed its published usage limits.
- Accessing, extracting, probing, or exfiltrating databases, private systems, non-public files, or personal data.
- Continuing collection after a website denies access or indicates that automated collection is prohibited.

Use conservative request rates, collect only the minimum public data needed, and respect applicable laws, website terms, robots policies, copyright, privacy, and personal-information requirements. If legality, authorization, website policy, data sensitivity, or the public status of information is unclear, stop and ask the user before proceeding. This boundary overrides all other project requirements.

# 更新记录

## 0.2.0 - 2026-06-24

- 新增面向安徽考生的招生资料助手
- 支持按院校、年份检索官方招生章程、在皖计划和录取资料
- 接入阳光高考公开招生章程入口和安徽省教育招生考试院入口
- 导出结果新增院校、省份、年份、资料类型和来源机构字段
- 明确不读取登录后数据，不提供录取概率预测

## 0.1.0 - 2026-06-24

- 首次公开版本
- 支持学校标识、校园通知和招生信息采集
- 支持 CSV、JSON 和 Markdown 导出
- 增加 robots.txt 检查、同站点限制和保守访问间隔
- 提供本地网页界面和 Windows 单文件打包脚本
